<div style="text-align: center;"><h1>Système de Gestion et Réservation -- UPVD</h1></div>
Application web développée avec Laravel permettant la gestion des salles, des matériels et des réservations au sein de l'université de Perpignan.

## Fonctionnalités
- **Administration**
  - Gestion des utilisateurs (ajout, modification, suppression)
  - Gestion des salles
  - Gestion des matériels
  - Tableau de bord administrateur
- **Réservations**
  - Création de réservations
  - Consultation des disponibilités 
  - Historique des réservations
## Technologies utilisées
  - Laravel (php)
  - Mysql
  - Blade (templates)
  - Bootstrap
  - Javascript
  - HTML/CSS
